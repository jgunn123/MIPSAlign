btToCounts {MIPSAlign}R Documentation

Convert bowtie output

Description

Convert bowtie or bowtie2 output to counts

Usage

btToCounts(df)

Arguments

df

Output dataframe from an aligner in MIPSAlign

Value

Two dataframes in a list that contain the aligned and unaligned reads, respectively

Examples

newdf <- btToCounts(df)
