btAlign {MIPSAlign}R Documentation

Bowtie aligner

Description

Generates counts using the original bowtie algorithm

Usage

btAlign(
  seqData = list.files(pattern = "*.fastq.gz$"),
  orfLib = list.files(pattern = "*.fa$|*.fasta$")[[1]],
  numCores = 1,
  keepFiles = FALSE,
  options = NULL
)

Arguments

seqData

The list of Fastq files you want to align

orfLib

The reference Fasta alignment file

numCores

The number of cores to run on

keepFiles

TRUE or FALSE, whether you want to keep the bowtie output files

options

Extra options for bowtie that feed into "alignmentParameter"

Value

A list of aligned and unaligned counts dataframes, respectively

Examples

x <- btAlign(orfLib = fastaFileName,numCores = 6)
