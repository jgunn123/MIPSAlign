WriteFasta {MIPSAlign}R Documentation

Write Fasta file

Description

Saves a dataframe as fasta file

Usage

WriteFasta(data, filename)

Arguments

data

The dataframe with "name" and "sequence" columns that you want stored

filename

The name of the file output

Value

A Fasta file in appropriate format

Examples

fa <- ReadFasta(getwd())
